# v-carousel
vertical text only carousel

## Example

```html
 <p> This text is static but this one 
            <span class='v-carousel'>
                <span>
                    <span>will </span>
                    <span>change</span>
                    <span>every</span>
                    <span>5 seconds</span>
                </span>
            </span>
        </p>
  ```
